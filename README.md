# Amigo secreto

Aplicación para realizar sorteo de nombres y saber quién será amigo secreto de quién...

## Ejecuta localmente

Clona el proyecto

```bash
git clone https://github.com/marianapatinom/ChallengeAmigoSecreto
```
Accede al archivo html desde cualquier navegador
